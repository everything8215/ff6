$00-$FF Direct Page
-------------------
   +$00 Always #$0000
   +$02 Always #$FFFF
   +$04 Buttons Pressed this Frame (unmapped)
   +$06 Buttons Pressed this Frame (mapped based on button config)
   +$08 Buttons Pressed this Frame but not Last Frame
   +$0A Buttons in Repeat Mode
   +$0C Buttons Pressed Last Frame
        axlr----
        a: A button down
        x: X button down
        l: L button down
        r: R button down
        byetudlr
        b: B button down
        y: Y button down
        e: Select button down
        t: Start button down
        u: Up direction down
        d: Down direction down
        l: Left direction down
        r: Right direction down
