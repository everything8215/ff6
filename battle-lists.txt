AI Script Commands (Monster Script)
------------------

Any command less than $F0 simply uses that attack
$EF uses the monster's special attack

F0 xx yy zz  $C21B22 Use attack xx, yy, or zz
F1 xx        $C21B47 Targetting (effect depends on value of xx)
                     00-2F: target actor xx
                     30-35: target monster slot xx
                     36: target self
                     37: target all monsters except self
                     38: target all monsters
                     39: target a random monster other than self
                     3A: target a random monster
                     3B: target all characters with wound status
                     3C: target a random character with wound status
                     3D: target all monsters with wound status
                     3E: target a random monsters with wound status
                     3F: target all characters with reflect status
                     40: target a random character with reflect status
                     41: target all monsters with reflect status
                     42: target a random monster with reflect status
                     43: target all characters
                     44: target a random character
                     45: target the last character who targetted you
                     46: target all characters and all monsters
                     47: target nothing
                     48: target character slot 1
                     49: target character slot 2
                     4A: target character slot 3
                     4B: target character slot 4
                     4C: target a random character or monster other than self OR target all characters and monsters except self (1/2 chance)
                     4D: target "Targetting" target
F2 xx yyyy   $C21B6E Do command $20, Change Battle to yyyy (monsters go back to full HP/MP if MSB set), monster exit effects xx (scroll battle background if MSB set)
F3 xx yy     $C21B66 Do command $21, Display short battle dialog xx
F4 xx yy zz  $C21B1C Do command xx, yy, or zz
F5 xx yy zz  $C21B57 Do command $24 (monster entrance/exit) zz is bitmask for affected monsters (zz = 0 means affect current monster only)
                     Value of xx
                     00: enter/exit instantly
                     01: enter/exit in a puff of smoke
                     02: fall from top/go out through top
                     03: come from sides, individually
                     04: jump out of/into water
                     05: float from ceiling
                     06: jump out of/into sand
                     07: come from sides, at the same time
                     08: fade in/out top to bottom
                     09: fade in/out bottom to top
                     0A: fade in/out (materialize)
                     0B: fade in/out (bottom left to top right, reversed for exit)
                     0C: boss death (can't be used for entrance)
                     0D: flash in/out
                     0E: Chadarnook (fade from one painting to the other)
                     0F: slow drop from above (final Kefka)
                     10: scroll battle background
                     11: final KEFKA death
                     Value of yy
                     00: $C24FE0 revive monsters and restore HP
                     01: $C24FCB hide monsters
                     02: $C24FEC revive monsters at current HP
                     03: $C24FCE hide monsters (can still be targetted)
                     04: $C24FC2 hide monsters but don't allow battle to end
                     05: $C24FFF hide monsters and set wound status (used at C2/643E only)
F6 xx yy zz  $C21B05 [xx = 0 Use, xx = 1 Throw] item yy (2/3 chance) or zz (1/3 chance)
F7 xx        $C21B8E Do command $23, Execute battle event xx
F8 xx yy     $C21B72 Do command $2E, Byte-wise battle variable manipulation (effect depends on top two bits of yy)
                     00: Set battle variable xx to yy
                     40: Set battle variable xx to yy (unused)
                     80: Add yy to battle variable xx
                     C0: Subtract yy from battle variable xx
F9 xx yy zz  $C21B76 Do command $2F, Bit-wise battle variable manipulation (effect depends on value of xx)
                     00: Toggle bit zz of battle variable yy
                     01: Set bit zz of battle variable yy
                     02: Clear bit zz of battle variable yy
FA xx yy zz  $C21BA0 Do command $2B (effect depends on value of xx) yy = target monster (flags)
                     00: flash monster red (Ultros/Umaro)
                     01: move monster back 8 pixels (slowly, Ultros)
                     02: move monster forward 8 pixels (slowly, Ultros)
                     03: move monster back 8 pixels (instantly, Pugs)
                     04: move monster forward 8 pixels (instantly, Pugs/Master Pug)
                     05: all characters run from left to right side of screen (unused)
                     06: all characters run from right to left side of screen (SrBehemoth)
                     07: move monster back 64 pixels (instantly, Ultros/Pugs/Master Pug)
                     08: move monster forward 64 pixels (instantly, unused)
                     09: play sound effect zz (pan yy, Telstar/Chaser/Dadaluma)
                     0A: final KEFKA disembodied head
                     0B: monsters glow (long, Atma)
                     0C: monsters glow (short, Atma)
                     0D: final KEFKA death animation
FB xx yy     $C21B6A Do command $30 (effect depends on value of xx)
                     00: $C21E67 reset monster timer
                     01: $C21E79 target yy becomes invincible
                     02: $C21E73 end battle immediately
                     03: $C21EA9 add GAU to party
                     04: $C21EC7 reset battle timer
                     05: $C21E87 target yy loses invincibility
                     06: $C21E95 target yy can be targetted
                     07: $C21E9F target yy cannot be targetted
                     08: $C21ECE fill target yy's ATB gauge
                     09: $C21E6F end battle after GAU returns from the veldt
                     0A: $C21F08 no effect
                     0B: $C21EE5 set status yy
                     0C: $C21EDA clear status yy
                     0D: $C21F00 set hide status (used during piranha battle)
FC xx yy zz  $C21A91 Conditional (xx determines the condition)
                     00: $C21D2C no condition (never continue)
                     01: $C21C40 if command yy or zz was used against the current monster
                     02: $C21C3F if attack yy or zz was used against the current monster
                     03: $C21C3B if item yy or zz was used against the current monster
                     04: $C21C5E if element yy (zz unused) was used against the current monster
                     05: $C21C70 if any attack was used against the current monster
                     06: $C21BB7 if target yy's HP is below (zz * 128)
                     07: $C21BC8 if target yy's MP is below (zz)
                     08: $C21BD7 if target yy has status zz
                     09: $C21C26 if target yy does not have status zz
                     0A: $C21D2C no condition (never continue)
                     0B: $C21C86 if monster timer has passed (yy * 2)
                     0C: $C21C9C if battle variable yy < zz
